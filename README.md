# sammieele-english391c

## AMAI Café Website

This repository documents my work in ENGLISH 391C: Intro to Web Design.  
It includes class exercises and the final AMAI website as the cumulative project. 

### About the Project

This website expands AMAI, a conceptual café brand I originally developed in ART 264: Digital Design for coursework and portfolio development.
In the branding project, I established the core visual and experiential language by designing:

- Logo
- Typography direction
- Color palette
- Patterns
- Branded collateral (menu, zarfs, and cups)

The final website extends that identity into a multi-page digital experience, serving as both a continuation of my ART 264 branding process and a case study in cross-platform brand design.

### Repository Structure

- Exercises: in-class practice files and checkpoints
- Final website: the polished AMAI site that brings the semester work together

### Purpose

This repo shows my process from foundational class exercises to a complete, brand-consistent final web project.
